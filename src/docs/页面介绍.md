# 页面介绍

## 功能

根据提供的字段进行课程查找，是 1 系统的增强版。

## 缘起

因为 1 系统的查询系统灵活度太低，无法满足需求，因此自己编写一个查询网站。

## 涉及的技术

前端 | 后端 | 数据库 |
-----|-----| ----- |
vue | flask | MySQL |

### 前端

使用 `vue` 框架，刚上手有些难度。

主要采用的技术有：

* `vue-routers` 进行页面跳转
* `CSS` 进行页面美化
* 一些方法，用来实现前端的内容筛选
* 根据表格的不同字段，向后端发送数据请求

### 后端

采用 `flask`

主要的功能是：

* 接收前端的请求，向数据库发送 `SQL` 语句
* 返回查找好的列表内容给前端

### 数据库

采用了 `MySQL`，共一个数据库，储存了一些表格，每个表格是一学期的所有课程。

* 对于从 1 系统导出的 `excel` 表格进行处理，需要先导出为 `.csv` 逗号分隔文件，再通过 `pandas` 库进行数据的优化整理。

## 可以改进的地方
